# 全能多变的完美绘者-ルリア老师

		> ルリア老师的知名作品很多，但是因为产量较低，所以本人的知名度没有跟上他的作品。据说他的每一幅图都要磨很久，可见他对自己的绘图是相当的认真。比起人物本身，ルリア老师更注重构图，细腻的表现手法，印象深刻的场景描绘，把人物与背景融为了一体，给人一种非常棒的视觉体验。
